# resume 你的专属简历（微信直接搜索即可）或者扫以下小程序二维码
@_@  教你前端如何一个人从零基础开发完整的小程序项目，包括后台可视化数据库，全程一个人即可独立完成。<br> 
@_@  该项目实现了个人简历的编辑、修改、展示、转发分享，以及查看他人简历，收藏他人简历，还有最近浏览过的简历。
